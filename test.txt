Задание для backend разработчика:

Общее описание:
Написать backend-service, который предоставляет API для взаимодействия с блокчейном на разных сетях (в базовом задании используется только Avalanche).

Базовае задание:
• Первый эндпоинт принимает номер блока для расчёта и адрес кошелька. Возвращает баланс на кошельке на момент переданного блока в сети Avalanche.
• Второй эндпоинт, принимающий на вход номер блока, возвращает все события произошедшие в смарт контракте 0x66357dCaCe80431aee0A7507e2E361B7e2402370 на сети Avalanche с переданного блока по текущий.

Бонусное задание (если захочется):
• Первый эндпоинт принимает дополнительный параметр - сеть. Следовательно и возвращает данные  по кошельку для переданной сети. (Можно реализовать для двух сетей Avalanche/Ethereum)

Требования к стеку и оформлению:
• Сервис должен быть написан на FastAPI, предполагается использование библиотеки web3.
• Валидация данных с помощью моделеи‌ Pydantic на вход и на выход.
• Должны быть написаны юнит тесты на pytest,
проверяющие правильность работы API.
• Собрать проект в Dockerfile
